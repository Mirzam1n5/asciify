# asciify

Simple Rust-based tool to convert images into ASCII art and export as '.png'. I did this to learn Rust :P. 

## Installation

You need to have [Rust](https://www.rust-lang.org/tools/install) installed.

```bash
cargo install --git https://github.com/Mirzam1n5/asciify
```

## Usage

asciify <image_path>
